<h1>Блок с поиском репозиториев github по подстроке</h1>

<h2>Блок содержит:</h2>
<ul>
<li>поле ввода (для подстроки поиска)</li>

<li>если по запросу ничего не найдено, то вместо списка сообщение "Ничего не найдено"</li>
</ul>
